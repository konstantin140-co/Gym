# Спортивный клуб (вариант 11)

WinForms + .NET 8 + EF Core + SQLite — отдельный проект по методичке ПП.01.

## UI (отличия от других вариантов)

| | Rental (12) | Deanery (7) | **Gym (11)** |
|---|---|---|---|
| Акцент | Зелёный | Синий | **Красный** |
| Навигация | Слева | Сверху | **Справа** |
| Списки | Карточки 3 в ряд | Строки | **Широкие карточки** |

## Функционал (вариант 11)

- **Клиент**, **Абонемент**, **Посещение** — CRUD
- Продажа абонемента (30 / 90 / 365 дней, расчёт даты окончания)
- Регистрация посещения (проверка активного абонемента)
- Истекающие абонементы (< 7 дней)
- Статистика посещений за период
- Список должников
- Поиск, фильтры, светлая/тёмная тема (Вид)

## Запуск

```powershell
cd F:\Develop\VS\proects\Gym
dotnet run --project Gym.Forms
```

## Тесты

```powershell
dotnet test Gym.Tests
```

## Структура solution

```
Gym.sln
├── Gym.Forms/     ← WinForms, UI, формы
├── Gym.Data/      ← модели, DbContext, SubscriptionCalculator
└── Gym.Tests/     ← NUnit
```

База данных: `gym.db` (создаётся автоматически при первом запуске).
